## Sobre o projeto

O **Aplicativo Agrovivaz** foi desenvolvido para facilitar a integrando das funcionalidades de visualização e intenção de compra / pedido das variedades.

## Features integradas

- **Integração com Whatsapp**: Utilização da API do Whatsapp para envio de mensagens com uma intenção de compra.

## Tecnologias Usadas

- **Flutter**
