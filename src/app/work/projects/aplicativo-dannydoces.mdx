## Sobre o projeto

O **Aplicativo Danny Doces** foi desenvolvido para facilitar a integração das funcionalidades de visualização e intenção de compra / pedido das produtos.

## Features integradas

- **Sacola de compras**: Adição de um ou mais produtos em uma sacola de compras para envio final via whatsapp.
- **Integração com Whatsapp**: Utilização da API do Whatsapp para envio de mensagens com uma intenção de compra.

## Tecnologias Usadas

- **Flutter**
