## Sobre o projeto

O **Aplicativo Gorgonzola Burger** foi desenvolvido para facilitar a integração das funcionalidades de visualização e intenção de compra / pedido das produtos.

## Features integradas

- **Integração com Whatsapp**: Utilização da API do Whatsapp para envio de mensagens com uma intenção de compra.

## Tecnologias Usadas

- **Flutter**
