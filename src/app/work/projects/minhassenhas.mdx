## Sobre o projeto

**Minhas Senhas 4** é um gerenciador de senhas mobile desenvolvido em Flutter que oferece autenticação biométrica integrada, armazenamento seguro com criptografia AES-256-CBC e suporte multiplataforma (Android, iOS e Web).

## 🔐 Features Principais

**Segurança**

  ✅ Criptografia AES-256-CBC de ponta a ponta<br />
  ✅ Autenticação biométrica (Face ID / Impressão Digital)<br />
  ✅ Armazenamento seguro com FlutterSecureStorage<br />
  ✅ Tokens JWT para autenticação com servidor<br />
  ✅ One-time protection para sincronização MySQL<br />
  ✅ SharedPreferences criptografadas<br />

**Autenticação Biométrica**

  ✅ Detecção automática de capacidade biométrica<br />
  ✅ Ativação pós-primeiro login<br />
  ✅ Fallback para login tradicional<br />
  ✅ Suporte para Face ID, Fingerprint e outros tipos<br />
  ✅ Armazenamento seguro de credenciais<br />

**Gerenciamento de Senhas**

  ✅ Adicionar, editar e deletar credenciais<br />
  ✅ Banco de dados SQLite local<br />
  ✅ Sincronização com servidor MySQL (opcional)<br />
  ✅ Persistência de sessão e preferências<br />
  ✅ Autenticação de usuários com email e senha<br />
  ✅ Interface intuitiva e responsiva<br />

## 🛠️ Tecnologias Usadas

- Flutter 3.7.2<br />
- Dart 3.x<br />
- SQLite (armazenamento local)<br />
- MySQL (sincronização opcional)<br />
- Local Auth (autenticação biométrica)<br />
- Flutter Secure Storage (armazenamento seguro)<br />
- AES-256-CBC (criptografia)<br />
- MobX (state management)<br />
- JWT (autenticação com servidor)<br />

## 📊 Stack Tecnológico

- **http**: Requisições HTTP
- **flutter_secure_storage**: Armazenamento encriptado de credenciais
- **local_auth**: Autenticação biométrica
- **sqflite**: Banco de dados SQLite
- **encrypt**: Criptografia AES
- **mobx / flutter_mobx**: State management
- **shared_preferences**: Armazenamento local
- **url_launcher**: Abrir URLs
- **file_picker**: Seletor de arquivos
- **intl**: Internacionalização
- **mysql1**: Driver MySQL

## 🔒 Arquitetura de Segurança

- **Biometria nativa**: LocalAuthentication do dispositivo
- **Armazenamento seguro**: FlutterSecureStorage para credenciais
- **Criptografia**: AES-256-CBC para senhas armazenadas
- **Tokens JWT**: Autenticação segura com servidor
- **MainActivity**: Estende FlutterFragmentActivity para suporte a biometria
