## Sobre o projeto

Um e-commerce de qualidade profissional construído como teste de aprendizado, containerizado com Docker para facilitar desenvolvimento e deployment.

## Features integradas

**Backend (Laravel)**

  ✅ API REST completa<br />
  ✅ Autenticação com JWT (Sanctum)<br />
  ✅ Admin Authentication System (v0.9) - Tabela separada de admins com sistema de roles<br />
  ✅ Modelos: Produtos, Categorias, Marcas, Cores, Destaques, Admins<br />
  ✅ Sistema de Pedidos (Orders & OrderItems)<br />
  ✅ Seeds com dados de exemplo<br />
  ✅ Gerenciamento de imagens<br />
  ✅ Configurações da loja<br />
  ✅ Dados de contato e métodos de pagamento<br />

**Frontend (React)**

  ✅ Página inicial com hero carousel<br />
  ✅ Grid de produtos com filtros<br />
  ✅ Página de detalhe do produto<br />
  ✅ Carrinho de compras funcional<br />
  ✅ Sistema de autenticação (usuários)<br />
  ✅ Admin Panel Dashboard (v0.9) - CRUD completo de produtos, categorias, pedidos e usuários<br />
  ✅ Admin Login - Autenticação separada com JWT<br />
  ✅ Minha conta e meus favoritos<br />
  ✅ Histórico de pedidos<br />
  ✅ 5 páginas de suporte (FAQ, Envio, Devoluções, Termos, Privacidade)<br />
  ✅ Suporte multilíngue (PT-BR/EN/ES)<br />
  ✅ Design totalmente responsivo<br />


## Tecnologias Usadas

- Laravel<br />
- ReactJs
