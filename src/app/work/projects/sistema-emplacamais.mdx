## Sobre o projeto

O **Sistema Emplaca Mais** foi desenvolvido para facilitar a integração de pedidos de placas pelos parceiros, clientes, concessionárias e lojas.

## Features integradas

- **Integração com Telegram**: Utilização da API do Telegram para envio de mensagens com as informações dos pedidos tanto para o administrador como para o instalador da placa.

## Tecnologias e Infra-estrutura utilizadas

- **Laravel**
- ** Servidor VPS na King Host**
