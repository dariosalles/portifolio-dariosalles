## Sobre o projeto

O **Website Agrovivaz** foi desenvolvido para facilitar  o usuário a conhecer a história sobre a Agrovivaz bem como integração das funcionalidades de interesse decompra, visualização, detalhes e mais informações sobre os mudas da espanha.

## Features integradas

- **Integração com Youtube**: Utilização dos vídeos do canal da Agrovivaz.
- **Envio de mensagem via form**: Envio de mensagem através do formulário direto para o canal de relacionamento com o cliente da Agrovivaz.

## Tecnologias Usadas

- **HTML**
- **CSS**
- **Javascript**
- **PHP**
